# 7 Segment Display Decoder using Boolean Algebra

This project is a combinational logic circuit that converts binary inputs into decimal outputs on a 7-segment display.

I built this project to understand how digital systems convert binary data into visible numbers using pure hardware logic.

---

## About the Project

For this project, I first created the truth table for the 7-segment display.  
The truth table defines which segments should turn ON or OFF for every binary input.

Using Boolean Algebra, I derived equations for each segment of the display.  
I solved the logic using both:

● SOP (Sum of Products)  
● POS (Product of Sums)

After simplifying the expressions, I selected the most optimized equations and used them to build the final circuit.
